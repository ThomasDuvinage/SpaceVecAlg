Detailed Description

template<typename T>
class sva::PTransform< T >

Pl√ºcker transform compact representation.

Use 3D matrix as rotation internal representation. Quaternion are inversed as they must be expressed in successor frame. See Roy Featherstone ¬´Rigid Body Dynamics Algorithms¬ª page 247.
